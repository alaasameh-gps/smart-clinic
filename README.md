# Smart Clinic System

A modern React + Tailwind UI prototype for a medical appointment system with patient and doctor/admin experiences.

## Features

- Landing page with hero messaging and benefit cards
- Login / registration flow with inline validation
- Patient dashboard showing upcoming appointments and quick actions
- Multi-step booking wizard with department, doctor, date, and time selection
- Real-time queue status panel with priority patient highlighting
- Doctor/admin appointment table with filters and patient detail modal
- Notifications, loading states, and mobile-first responsive layout

## Local Setup

1. Open a terminal in `smart-clinic-system`
2. Run `npm install`
3. Run `npm run dev`
4. Open the displayed Vite URL in a browser

## Notes

- The app uses mock data and simulated API calls for interactive behavior.
- The UI is designed for clean medical styling with blue-white palette, rounded cards, and soft shadows.
